연호와 서력,
즉시 변환.

레이와・헤이세이・쇼와 등 일본 연호와 서력을 한 번에 변환. 간지・로쿠요・절기로 일본 전통 달력을 일상 속에서 만나보세요.

App Store에서 다운로드

Reki로 할 수 있는 것

🔄

연호 ⇄ 서력 변환

레이와・헤이세이・쇼와・다이쇼・메이지 지원. 입력하면 즉시 변환. 일본 서류 확인에 최적.

🐉

간지・로쿠요・절기

오늘의 간지, 로쿠요 길흉, 현재 절기를 한눈에 확인. 이십사절기와 로쿠요 상세 설명 내장.

📷

카메라 인식

연호가 포함된 문서를 촬영하면 자동으로 인식하고 변환. 수동 입력 불필요.

🎙️

음성 입력

날짜를 말하면 바로 변환. 5개 언어 음성 인식 지원.

🔲

홈 화면 위젯

오늘의 연호・간지・로쿠요를 홈 화면이나 잠금 화면에 상시 표시. 계절에 맞게 배경 자동 변경.

🌍

5개 언어 지원

일본어・영어・간체 중국어・번체 중국어・한국어 완전 지원. 방일 여행과 업무 모두 편리.
